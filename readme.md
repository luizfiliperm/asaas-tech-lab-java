# asaas-tech-lab-java

Participe do Hackathon Asaas Tech Lab, um desafio técnico que utiliza uma API especialmente desenvolvida para o evento.

Durante o hackathon, você terá duas horas para resolver problemas e criar soluções inovadoras, competindo por prêmios como MacBooks, PlayStation 5 e muito mais.

É uma excelente oportunidade para demonstrar suas habilidades técnicas e inovadoras enquanto concorre a grandes recompensas.

## Como subir esse projeto

1. Clone o repositório
2. Abra o projeto em sua IDE de preferência
3. Execute o comando abaixo para gerar o artefato: 
```shell
mvn clean package
```
4. Execute o comando abaixo para subir o projeto:
```shell
docker-compose up
```

Você também pode rodar o projeto através dos atalhos da sua IDE, como por exemplo, o atalho `Run` e `Debug` do IntelliJ IDEA.
